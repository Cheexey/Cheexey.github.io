Devlog project 3DWorld

Voor dit project is de opdracht om een subscriptie gebaseerde website te maken,
waarop mensen 3Dprint-artiesten kunnen steunen DMV van een maandelijkse bijdrage.
In deze devlog documenteer ik mijn werk voor dit project.

Website template
(Datum van toepassing: 19-10-23)


Om dit project soepel te laten verlopen, is het natuurlijk de bedoeling dat alle pagina's er hetzelfde uit zien.
Omdat ik met meerdere individuelen aan dit project werk, is het dus handig als er een globale template beschikbaar is zodat iedereen in iedergeval in grote lijnen dezelfde layout heeft voor zijn pagina.
Daarom heb ik voor het hele team een responsive en dynamische template gemaakt die we allemaal kunnen toepassen op onze pagina's.

Deze webpagina bestaat uit een "sticky" navbar, met daarin hyperlinks naar:
-De homepage
-De discovery sectie
En een dropdown menu voor:
-Uitloggen
-Instellingen
-Meer algemene accountinformatie

Daaronder een:
Pagina content sectie
Een footer met hyperlinks naar andere externe pagina's.
